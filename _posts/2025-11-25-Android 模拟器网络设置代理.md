# 列出所有模拟器名称（
```shell
emulator -list-avds  # 列出所有模拟器名称（如：Pixel_5_API_33）

```

# 执行代理命令启动
```shell
emulator -avd 模拟器名称 -http-proxy http://代理IP:端口
# 示例：emulator -avd Pixel_5_API_33 -http-proxy http://127.0.0.1:8888
```
